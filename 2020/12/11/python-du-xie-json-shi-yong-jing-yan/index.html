目前在做的工作，对接需要用JSON格式，所以就记录了一些内容。

  1. 爬虫任务，需要把爬下来的数据以json格式转给对方。

在scrapy框架下，写入json格式的内容，保存.json文件

class JsonPipeline(object):
    def process_item(self, item, spider):
        base_dir = 'result' + os.sep + item['keyword']
        if not os.path.isdir(base_dir):
            os.makedirs(base_dir)
        jsonfile_path = base_dir + os.sep + item['keyword'] + '.json'

        if item:
            with codecs.open(jsonfile_path, 'a+') as f:
                data = {
                'id': item['weibo']['user_id'],
                'bid': item['weibo']['bid'],
                'screen_name': item['weibo']['screen_name'],
                'pics': item['weibo']['pics']
                }

                lines = json.dumps(data, ensure_ascii=False)
                f.write(lines + "\n")

        return item

json_bigdata = []

            for i in range(len(json_data["data"])):
